Description du projet :
Le projet Arduino consiste à la manipulation de la carte électronique Arduino UNO. C’ est une carte électronique OpenSource à base de microcontrôleurs. Ce dernier peut être programmé pour analyser et produire des signaux électriques en vue de mettre en œuvre des applications diverses comme le pilotage de robot, la domotique, le prototypage rapide etc...
Nous avons choisis pour ce projet de créer une boussole électronique. Le but de ce projet est de concevoir une boussole à l’aide des informations récupérées depuis le magnétomètre du Shield 9-Axis Motion. Ces données seront traitées pour capter le champ magnétique terrestre et calculer l’orientation du composant par rapport au nord géomagnétique. Le résultat de ce calcul devra être formaté pour pouvoir être affiché en temps réel et de manière ludique sur le Shield LCD.
